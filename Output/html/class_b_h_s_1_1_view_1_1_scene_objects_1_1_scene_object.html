Detailed Description

Абстрактный базовый класс визуального объекта сцены.

SceneObject определяет общие свойства и поведение всех визуальных элементов, отображаемых на сцене — таких как Ball и Wall.

Содержит информацию о позиции, форму для отображения, а также методы для обновления цвета и визуализации. Обновления цвета выполняются в UI-потоке Avalonia через Dispatcher.UIThread.

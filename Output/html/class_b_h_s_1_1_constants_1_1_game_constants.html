Detailed Description

Набор глобальных игровых констант, используемых в симуляции.

Класс GameConstants содержит базовые значения, определяющие скорость обновления игрового цикла и шаг времени симуляции.

Эти параметры используются всеми системами ECS для расчёта перемещения и таймингов.

Detailed Description

Класс инициализации и управления ECS-миром.

EcsStartup создаёт ECS-мир (EcsWorld) и регистрирует все игровые системы, управляющие логикой симуляции.

После инициализации через метод Initialize выполняет обновление систем в игровом цикле методом Run, а при завершении — освобождает ресурсы.

Инициализирует ECS-мир и регистрирует все игровые системы.

Список добавляемых систем:

  • MoveSystem — перемещение сущностей;
  • CheckCollisionSystem — проверка столкновений;
  • CollisionHandlingSystem — обработка отражений;
  • ColorChangeEventHandlingSystem — реакция на изменение цвета;
  • ColorChangeAnimationSystem — анимация возврата цвета;
  • PositionSyncSystem — синхронизация ECS с визуальной сценой.

Detailed Description

Фабрика для создания стен в ECS-мире и их визуальных представлений в сцене.

WallFactory инициализирует ECS-сущность, описывающую стену, — добавляет компонент EdgeComponent с координатами начала и конца отрезка, а также компонент ColorAnimationComponent для обработки анимации цвета.

После создания ECS-компонентов фабрика создаёт визуальный объект Wall, связывает его с сущностью через LinkToSceneObject и добавляет на сцену.

Метод добавляет в ECS-мир сущность с компонентами:

  • EdgeComponent — геометрическое описание стены;
  • ColorAnimationComponent — данные для цветовой анимации;
  • LinkToSceneObject — связь с объектом сцены.

Также вычисляет центр стены для корректного позиционирования визуального объекта.

Detailed Description

Класс, выполняющий регистрацию всех сервисов и фабрик приложения.

AppStartup конфигурирует контейнер зависимостей с помощью библиотеки Microsoft.Extensions.DependencyInjection.

В рамках инициализации регистрируются:

  • MainWindow — главное окно приложения;
  • EcsWorld — ECS-мир;
  • ISceneService (реализация Scene) — сервис управления сценой;
  • IEcsService (реализация EcsStartup) — управление ECS-системами;
  • BallFactory и WallFactory — фабрики для создания объектов;
  • SceneStartup — инициализация стартовой сцены.

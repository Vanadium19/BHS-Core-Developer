Detailed Description

Сцена, содержащая визуальные объекты и управляющая их отрисовкой.

Класс Scene реализует интерфейс ISceneService и отвечает за управление жизненным циклом визуальных объектов сцены — их добавлением, отображением и удалением.

Все операции визуального обновления выполняются в UI-потоке Avalonia через Dispatcher.UIThread для обеспечения потокобезопасности.

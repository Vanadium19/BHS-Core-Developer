Detailed Description

Компонент, описывающий геометрию стены в виде отрезка.

Хранит координаты начала (Start) и конца (End) отрезка, определяющего положение стены на сцене.

Также предоставляет метод GetNormal для вычисления нормали к поверхности стены, используемой при расчёте отражений.

Detailed Description

Система проверки столкновений между шариками и стенами в ECS-мире.

CheckCollisionSystem определяет факт столкновения каждого шарика (сущности с компонентами PositionComponent и RadiusComponent) со всеми стенами (сущностями, содержащими EdgeComponent).

При обнаружении пересечения создаётся событие CollisionEvent для шарика и ColorChangeEvent для стены.

Detailed Description

Класс, выполняющий инициализацию сцены и создание игровых объектов.

SceneStartup отвечает за построение начального состояния сцены — создаёт все стены и один шарик с параметрами из ObjectsData.

Использует фабрики WallFactory и BallFactory для добавления соответствующих ECS-сущностей и их визуальных представлений на сцену.

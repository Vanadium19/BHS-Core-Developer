Detailed Description

Интерфейс сервиса управления визуальной сценой.

Определяет базовые операции для добавления, отображения и удаления объектов сцены (SceneObject).

Реализация интерфейса обеспечивает взаимодействие ECS-механизма с визуальным представлением в Avalonia.

\doxysubsection{Detailed Description}
Система ECS, отвечающая за обновление позиции сущностей во времени. 

\doxylink{struct_b_h_s_1_1_systems_1_1_move_system}{Move\+System} проходит по всем сущностям, у которых есть компоненты \doxylink{struct_b_h_s_1_1_components_1_1_position_component}{Position\+Component}, \doxylink{struct_b_h_s_1_1_components_1_1_speed_component}{Speed\+Component} и \doxylink{struct_b_h_s_1_1_components_1_1_direction_component}{Direction\+Component}. 

На каждом кадре обновляет позицию сущности по формуле\+: {\ttfamily position += direction \texorpdfstring{$\ast$}{*} speed \texorpdfstring{$\ast$}{*} Delta\+Time}, перемещая объекты в соответствии с их направлением, скоростью и временем кадра.

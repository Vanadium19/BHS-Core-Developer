\doxysubsection{Detailed Description}
Структура, описывающая параметры шарика для инициализации в ECS-\/мире. 

Используется при создании сущностей шаров через \doxylink{class_b_h_s_1_1_factories_1_1_ball_factory}{BHS.\+Factories.\+Ball\+Factory}.

\doxysubsection{Detailed Description}
Класс, выполняющий регистрацию всех сервисов и фабрик приложения. 

\doxylink{class_b_h_s_1_1_core_1_1_app_startup}{App\+Startup} конфигурирует контейнер зависимостей с помощью библиотеки Microsoft.\+Extensions.\+Dependency\+Injection. 

В рамках инициализации регистрируются\+: 
\begin{DoxyItemize}
\item \doxylink{class_b_h_s_1_1_main_window}{Main\+Window} — главное окно приложения; 
\item Ecs\+World — ECS-\/мир; 
\item \doxylink{interface_b_h_s_1_1_view_1_1_i_scene_service}{IScene\+Service} (реализация \doxylink{class_b_h_s_1_1_view_1_1_scene}{Scene}) — сервис управления сценой; 
\item \doxylink{interface_b_h_s_1_1_core_1_1_i_ecs_service}{IEcs\+Service} (реализация \doxylink{class_b_h_s_1_1_core_1_1_ecs_startup}{Ecs\+Startup}) — управление ECS-\/системами; 
\item \doxylink{class_b_h_s_1_1_factories_1_1_ball_factory}{Ball\+Factory} и \doxylink{class_b_h_s_1_1_factories_1_1_wall_factory}{Wall\+Factory} — фабрики для создания объектов; 
\item \doxylink{class_b_h_s_1_1_core_1_1_scene_startup}{Scene\+Startup} — инициализация стартовой сцены. 
\end{DoxyItemize}

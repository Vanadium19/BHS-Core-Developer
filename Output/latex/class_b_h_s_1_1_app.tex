\doxysubsection{Detailed Description}
Главный класс приложения, отвечающий за инициализацию и запуск игрового цикла. 

\doxylink{class_b_h_s_1_1_app}{App} наследуется от Application Avalonia и управляет жизненным циклом всего приложения. В процессе запуска создаёт контейнер зависимостей, регистрирует все сервисы через \doxylink{class_b_h_s_1_1_core_1_1_app_startup}{App\+Startup} и инициализирует ECS-\/мир и визуальную сцену. 

После инициализации запускается асинхронный игровой цикл, который выполняет\+: 
\begin{DoxyItemize}
\item обновление логики ECS через \doxylink{interface_b_h_s_1_1_core_1_1_i_ecs_service}{IEcs\+Service}; 
\item отрисовку объектов сцены через \doxylink{interface_b_h_s_1_1_view_1_1_i_scene_service}{IScene\+Service}; 
\item задержку между кадрами, определяемую Game\+Constants.\+Sleep\+Time. 
\end{DoxyItemize}

\doxysubsection{Detailed Description}
Фабрика для создания сущностей шариков в ECS-\/мире и их визуальных представлений. 

\doxylink{class_b_h_s_1_1_factories_1_1_ball_factory}{Ball\+Factory} отвечает за инициализацию всех необходимых компонентов ECS-\/сущности, представляющей шарик, а также за создание связанного визуального объекта \doxylink{class_b_h_s_1_1_view_1_1_scene_objects_1_1_ball}{Ball} в сцене. 

Компоненты, добавляемые при создании\+:

\item \doxylink{struct_b_h_s_1_1_components_1_1_speed_component}{Speed\+Component} — скорость движения; 
\item \doxylink{struct_b_h_s_1_1_components_1_1_direction_component}{Direction\+Component} — направление движения; 
\item \doxylink{struct_b_h_s_1_1_components_1_1_radius_component}{Radius\+Component} — радиус шарика;

\end{DoxyItemize}

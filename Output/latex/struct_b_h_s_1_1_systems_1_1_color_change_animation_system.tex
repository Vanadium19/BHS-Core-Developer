\doxysubsection{Detailed Description}
Система анимации изменения цвета объектов сцены. 

\doxylink{struct_b_h_s_1_1_systems_1_1_color_change_animation_system}{Color\+Change\+Animation\+System} управляет постепенным возвращением цвета объекта к исходному после события \doxylink{struct_b_h_s_1_1_events_1_1_color_change_event}{BHS.\+Events.\+Color\+Change\+Event}. 

Для каждой сущности, имеющей \doxylink{struct_b_h_s_1_1_components_1_1_link_to_scene_object}{Link\+To\+Scene\+Object} и \doxylink{struct_b_h_s_1_1_components_1_1_color_animation_component}{Color\+Animation\+Component}, система уменьшает таймер анимации ({\ttfamily Current\+Time}) на каждом кадре и по его завершении восстанавливает начальный цвет ({\ttfamily Start\+Color}).

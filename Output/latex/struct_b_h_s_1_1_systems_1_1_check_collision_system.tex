\doxysubsection{Detailed Description}
Система проверки столкновений между шариками и стенами в ECS-\/мире. 

\doxylink{struct_b_h_s_1_1_systems_1_1_check_collision_system}{Check\+Collision\+System} определяет факт столкновения каждого шарика (сущности с компонентами \doxylink{struct_b_h_s_1_1_components_1_1_position_component}{Position\+Component} и \doxylink{struct_b_h_s_1_1_components_1_1_radius_component}{Radius\+Component}) со всеми стенами (сущностями, содержащими \doxylink{struct_b_h_s_1_1_components_1_1_edge_component}{Edge\+Component}). 

При обнаружении пересечения создаётся событие \doxylink{struct_b_h_s_1_1_events_1_1_collision_event}{Collision\+Event} для шарика и \doxylink{struct_b_h_s_1_1_events_1_1_color_change_event}{Color\+Change\+Event} для стены.

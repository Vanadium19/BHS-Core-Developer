\doxysubsection{Detailed Description}
Главное окно приложения, содержащее визуальные элементы сцены. 

\doxylink{class_b_h_s_1_1_main_window}{Main\+Window} является корневым контейнером пользовательского интерфейса Avalonia и содержит область сцены ({\ttfamily Canvas\+Root}), где располагаются визуальные объекты — \doxylink{class_b_h_s_1_1_view_1_1_scene_objects_1_1_ball}{BHS.\+View.\+Scene\+Objects.\+Ball} и \doxylink{class_b_h_s_1_1_view_1_1_scene_objects_1_1_wall}{BHS.\+View.\+Scene\+Objects.\+Wall}. 

Предоставляет доступ к коллекции элементов управления через свойство \doxylink{class_b_h_s_1_1_main_window_afe8da03f4c464d27f4744cc5db1ba360}{Controls}, используемое сценой (\doxylink{class_b_h_s_1_1_view_1_1_scene}{BHS.\+View.\+Scene}) для добавления и удаления объектов.

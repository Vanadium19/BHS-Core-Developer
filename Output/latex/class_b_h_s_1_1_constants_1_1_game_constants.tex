\doxysubsection{Detailed Description}
Набор глобальных игровых констант, используемых в симуляции. 

Класс \doxylink{class_b_h_s_1_1_constants_1_1_game_constants}{Game\+Constants} содержит базовые значения, определяющие скорость обновления игрового цикла и шаг времени симуляции. 

Эти параметры используются всеми системами ECS для расчёта перемещения и таймингов.

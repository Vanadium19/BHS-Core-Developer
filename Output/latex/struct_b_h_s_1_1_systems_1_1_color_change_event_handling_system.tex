\doxysubsection{Detailed Description}
Система обработки событий изменения цвета у объектов сцены. 

\doxylink{struct_b_h_s_1_1_systems_1_1_color_change_event_handling_system}{Color\+Change\+Event\+Handling\+System} реагирует на события \doxylink{struct_b_h_s_1_1_events_1_1_color_change_event}{Color\+Change\+Event}, обновляя визуальное состояние объектов сцены, связанных через компонент \doxylink{struct_b_h_s_1_1_components_1_1_link_to_scene_object}{Link\+To\+Scene\+Object}. 

При получении события система применяет цветовую анимацию, устанавливает целевой цвет из \doxylink{struct_b_h_s_1_1_components_1_1_color_animation_component}{Color\+Animation\+Component} и удаляет обработанное событие из ECS-\/мира.

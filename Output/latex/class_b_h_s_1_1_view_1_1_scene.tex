\doxysubsection{Detailed Description}
Сцена, содержащая визуальные объекты и управляющая их отрисовкой. 

Класс \doxylink{class_b_h_s_1_1_view_1_1_scene}{Scene} реализует интерфейс \doxylink{interface_b_h_s_1_1_view_1_1_i_scene_service}{IScene\+Service} и отвечает за управление жизненным циклом визуальных объектов сцены — их добавлением, отображением и удалением. 

Все операции визуального обновления выполняются в UI-\/потоке Avalonia через Dispatcher.\+UIThread для обеспечения потокобезопасности.

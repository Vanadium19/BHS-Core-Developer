\doxysubsection{Detailed Description}
Абстрактный базовый класс визуального объекта сцены. 

\doxylink{class_b_h_s_1_1_view_1_1_scene_objects_1_1_scene_object}{Scene\+Object} определяет общие свойства и поведение всех визуальных элементов, отображаемых на сцене — таких как \doxylink{class_b_h_s_1_1_view_1_1_scene_objects_1_1_ball}{Ball} и \doxylink{class_b_h_s_1_1_view_1_1_scene_objects_1_1_wall}{Wall}. 

Содержит информацию о позиции, форму для отображения, а также методы для обновления цвета и визуализации. Обновления цвета выполняются в UI-\/потоке Avalonia через Dispatcher.\+UIThread.
